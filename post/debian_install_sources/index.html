Debian安装时换源方法

· 2024-04-14 · # System # Maintain # Utility

问题起因

Debian安全源地址会使用Debian的官方国外源，非常的慢，把它换掉会好很多

解决方案

在安装步骤进入到选择安装的桌面环境和软件时, 键入Ctrl+Alt+F2 进入tty命令终端Enter

修改软件源头

nano /target/etc/apt/sources.list

Ctrl+X 退出保存

退出终端重新进入界面继续安装，Ctrl+Alt+F5
